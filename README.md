# Install
**Archlinux, Manjaro, Antergos (AUR)**:
```
yaourt -S papirus-icon-theme-gtk
```
**Ubuntu 14.04/16.04**:
```
sudo add-apt-repository ppa:varlesh-l/papirus-pack
sudo apt-get update
sudo apt-get install papirus-gtk-icon-theme 
```
